== 설치 방법 ==
1. 플러그인 폴더를 /wp-content/plugins/ 디렉토리에 업로드합니다.
2. WordPress의 '플러그인' 메뉴에서 플러그인을 활성화합니다.

== 설명 ==  
Heading ID Copy는 경량의 WordPress 플러그인으로, 자동으로 heading 태그(h2-h6)에 고유한 ID를 부여합니다.  
또한, 각 heading 옆에 작은 복사 아이콘을 추가하여 사용자가 해당 heading의 ID가 포함된 전체 URL을 빠르게 클립보드로 복사할 수 있도록 합니다.

### 기능  
- `h2-h6` 요소에 대해 자동으로 고유 ID를 생성합니다.  
- 각 heading 옆에 작은 복사 버튼을 추가합니다.  
- 버튼을 클릭하면 해당 heading ID가 포함된 전체 URL을 클립보드로 복사합니다.  
- 포스트와 페이지 모두에서 작동합니다.  
- 간단하고 경량으로 설계되어 성능에 미치는 영향이 최소화되어 있습니다.
